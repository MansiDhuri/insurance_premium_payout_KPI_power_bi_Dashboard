# 📊 Insurance Premium & Payout KPI Dashboard

## 📌 Project Overview

This project focuses on building an interactive Insurance Policy Analysis Dashboard using Microsoft Power BI. It provides a unified view of the insurance business by analyzing the complete policy lifecycle — from purchase to maturity.
The dashboard enables stakeholders to track performance, monitor claims, and gain insights into customer and policy behavior.

## 🎯 Business Objectives

- Track performance of different insurance policy types
- Analyze premium collection and payout trends
- Identify top-performing and underperforming policies
- Monitor claim settlement efficiency
- Understand customer distribution across states and segments
- Forecast maturity amounts and business growth
- Automate report refresh for real-time insights

## 🏢 About the Organization

TruSecure CreDebit Insurance provides innovative credit/debit-linked insurance solutions for individuals and businesses. The company focuses on:

Affordable premiums
- Fast claim settlements
- Transparent processes
- Technology-driven solutions

## 📂 Dataset Description

The dataset covers the entire insurance lifecycle (2015–2025) and includes:

### 👤 Customer Details

- Customer ID
- Name
- Gender
- Age
- Occupation
- Smoker Status
- Medical Requirement
- Nationality & State

### 📄 Policy Information

- Policy Number & Status
- Start Date & Last Paid Date
- Tenure (Years)
- Premium Amount & Payment Frequency
- Sum Assured / Coverage
- Maturity Amount & Surrender Value
- Loan Eligibility

### 🧾 Policy Classification

- Policy Type (Term, Whole Life, ULIP)
- Policy Name & Codes

### 🧑‍💼 Sales Agent Details

- Agent Code & Name
- Operating State

### 📊 Key KPIs

- 💰 Total Premium Amount
- 📈 Total Annual Premium
- 📊 Number of Policies
- 💵 Total Premium Paid
- 💳 Total Premium Payable
- 📉 ROI (%)
- 📈 CAGR (%)
- 💹 Profit / Gain

## 📌 Key Features of Dashboard

- 📍 State-wise performance analysis
- 👥 Customer segmentation insights
- 📊 Policy performance tracking
- 📅 Maturity forecasting (5, 10, 15, 20 years)
- 📉 Premium paid vs payable comparison
- 🔍 Dynamic filtering using flags
